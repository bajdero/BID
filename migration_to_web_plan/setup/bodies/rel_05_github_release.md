## Problem Statement

The official GitHub release 2.0.0 must be published with a complete changelog so that
users and integrators know what changed since 1.0.0 (the last desktop release).

## Scope

**In scope:**
- GitHub Release `2.0.0` created from tag `2.0.0`
- Changelog covering all features from M1–M13
- Release marked as latest (replacing 1.0.0)
- Migration guide from 1.0.0 (desktop) to 2.0.0 (web) linked

**Out of scope:**
- Binary artefacts for the desktop app (superseded by web)

## Acceptance Criteria
